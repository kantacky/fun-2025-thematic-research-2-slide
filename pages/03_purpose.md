# 目的

- 個人的穴場フォトスポットの抽出に必要なフォトウォークのデータを手に入れる

# 手段

- 大規模言語モデル (LLM) によるデータ生成

# 手法

- フォトウォークデータに求める要件を言語化
- Gemini API の構造化出力機能を用いてフォトウォークデータ生成
